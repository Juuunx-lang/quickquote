# QuickQuote 后端流程图

更新日期：2026-06-23

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant API as FastAPI
    participant W as ValuationWorkflow
    participant P as purchase_records
    participant J as 聚水潭
    participant S as supplier_quote SQLite

    U->>F: 输入商品 / 上传文件
    F->>API: POST /api/v1/chat/stream
    API->>W: 创建工作流输入
    W->>W: extract_stage
    W->>P: 查询采购候选
    W->>J: 查询商品、库存、入库成本
    W->>S: 查询供应商报价
    P-->>W: 采购历史候选
    J-->>W: 聚水潭候选
    S-->>W: 报价候选
    W->>W: 合并、去重、排序、补齐
    W-->>API: items/candidates
    API-->>F: SSE done
    F-->>U: 卡片展示、筛选、导出
```

## 节点说明

`extract_stage`：

- 解析商品信息。
- 支持 LLM 和规则兜底。

`multi_source_match_stage`：

- 三源并行召回。
- 单源失败不终止整个流程。

`purchase_route_stage`：

- 采购历史补齐。
- 聚水潭库存/成本补齐。
- 动态成本补齐。

`result_stage`：

- 输出最终结构。
- 保存上下文和归档。
