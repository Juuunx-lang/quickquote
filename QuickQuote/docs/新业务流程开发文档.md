# QuickQuote 新业务流程当前版

更新日期：2026-06-23

## 1. 业务目标

工作人员输入商品需求后，系统返回多条可复核候选。每条候选尽量聚合：

- 基本商品信息
- 历史采购信息
- 聚水潭库存和成本
- 供应商报价

最终由工作人员选择合适候选并导出 Excel。

## 2. 输入支持

- 单行商品
- 多行商品
- Excel 上传后拆行
- 图片/文件辅助解析
- 编码模糊匹配开关

## 3. 输出分组

前端左侧三个入口：

- 候选记录：全部候选
- 采购记录：采购表或聚水潭存在记录
- 报价列表：供应商报价表存在记录

## 4. 结果卡片

顶部基本字段：

- 产品名
- 品牌
- 型号（sku_id）
- 规格

详情来源：

- 最新采购信息
- 聚水潭来源
- 供应商报价

## 5. 采购信息

展示字段：

- 日期
- 数量
- 采购价
- 售价
- 结算单价
- 结算金额
- 供应商
- 店铺名

外购链接区域保留蒙版式入口，用于未来接入自动拉取外购链接。

## 6. 聚水潭信息

展示字段：

- 库存
- 最新成本价

最新成本价悬停提示显示成本价日期。成本波动按钮进入近 30 天采购入库明细。

## 7. 供应商报价

展示字段：

- 供应商
- 报价
- 更新时间

供应商报价来源可以单独返回候选。

## 8. Excel 导出

用户可以选择：

- 哪些候选记录
- 哪些具体产品
- 哪些字段

导出文件为 `.xlsx`，一行对应一个候选产品。
